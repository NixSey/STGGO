# STGGO
**ST**umble Guys Mass Account **G**rabber in **Go**-Lang.

# How To Use
### Run-Only
`go run main.go`

or

`go run main-cli.go <threads>` (CLI Version).

### Build-Only
`go build -o stggo`

or

`go build main-cli.go` (CLI Version).

then

`./sggo`

or

`./main-cli <threads>` (CLI Version).
