# STGGO
**ST**umble Guys Mass Account **G**rabber in **Go**-Lang.

# How To Use
### Run-Only
`go run main.go`

or

`go run main-cli.go <threads>`

### Build-Only
`go build -o stggo`

or

`go build <filename>.go`

then

`./<filename> <threads>`

or

`<filename>.exe <threads>` (Windows).

#### Note: Remove < > Instead.
